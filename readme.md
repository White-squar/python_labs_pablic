## 5_1.py — Двумерный массив, задачи 1–3

**Задача 1. Сумма и произведение нечётных отрицательных элементов**

Матрица создаётся генератором списков — двойным: внешний цикл по строкам, внутренний по столбцам. Для поиска нужных элементов используется list comprehension с тремя условиями одновременно: элемент должен быть отрицательным (`x < 0`), нечётным (`x % 2 != 0`) и по абсолютной величине меньше номера строки (`abs(x) < i+1`). Здесь `i+1` потому что в условии задачи строки нумеруются с единицы, а Python нумерует с нуля. Сумма считается встроенной функцией `sum()`, произведение — `math.prod()`. Если подходящих элементов нет, `prod()` вернёт ошибку, поэтому стоит проверка `if elems else`.

**Задача 2. Строки без отрицательных элементов**

Строка подходит, если все её элементы неотрицательны. Для этого используется встроенная функция `all()` — она возвращает `True` только если каждый элемент удовлетворяет условию. Это эффективнее чем считать отрицательные вручную: `all()` прерывается при первом же несоответствии.

**Задача 3. Минимум под побочной диагональю**

Побочная диагональ — это элементы, у которых сумма индексов равна `N-1`. Элементы ПОД побочной диагональю имеют `i + j > N - 1` (правый нижний треугольник). Из них фильтруем только положительные и берём `min()`. Если таких элементов нет — защита через `if`.

## 5_2.py — Операции со строками и столбцами

**Задача 1. Перестановка столбцов**

Ищем последний столбец, содержащий хотя бы один ноль. `any()` — противоположность `all()`: возвращает `True` если хотя бы один элемент удовлетворяет условию. Среди всех подходящих столбцов берём последний через `max()` по индексу. `default=None` защищает от ошибки если нулей в матрице нет вообще. Перестановка столбцов делается через Python-свап в одну строку: `a, b = b, a` — без временной переменной.

**Задача 2. Удаление строки**

Ищем первую строку с суммой `<= K`. Используется `next()` с генератором — это принципиально эффективнее чем собирать все подходящие строки в список: `next()` останавливается сразу при нахождении первого совпадения. `pop(idx)` удаляет строку по индексу, все последующие строки автоматически сдвигаются.

**Задача 3. Вставка столбца нулей**

Ищем последний столбец, где все элементы отрицательны — снова `all()` + `max()`. После нахождения вставляем столбец из нулей: `insert(pos, val)` в каждой строке на позицию `last_neg_col + 1` (то есть сразу после найденного столбца).

## 5_3.py — Квадратная матрица

**Задача 1. Элементы над побочной диагональю**

Побочная диагональ квадратной матрицы N×N — это элементы где `i + j == N - 1`. НАД ней — `i + j < N - 1`, это левый верхний треугольник. Собираем все такие элементы в одномерный список одним list comprehension с двойным циклом.

**Задача 2. Сортировка столбцов**

Ключевой приём — `zip(*A)` для обхода по столбцам. Оператор `*` распаковывает матрицу: вместо одного аргумента-списка `zip` получает строки как отдельные аргументы. `zip` объединяет i-е элементы каждой строки — получаются столбцы. `enumerate` даёт одновременно индекс `j` и сам столбец `col`. Если сумма столбца делится на 3 или 7 — сортируем через `sorted()` (он не меняет оригинал, а создаёт новый список) и записываем значения обратно в матрицу по индексу строки.

---

## Общие приёмы, которые стоит уметь объяснить

| Приём | Зачем |
|---|---|
| `[[...] for _ in range(N)]` | Создание матрицы генератором |
| `all()` / `any()` | Проверка условия по всей строке/столбцу |
| `max(..., default=None)` | Последний подходящий индекс без ошибки |
| `next(..., None)` | Первый подходящий элемент, остановка сразу |
| `zip(*A)` | Транспонирование / обход по столбцам |
| `a, b = b, a` | Свап без временной переменной |


## 6
```python
is_num = lambda w: bool(re.fullmatch(r"-?\d+(\.\d+)?", w))
```
`lambda` — это функция в одну строку. Принимает слово `w` и проверяет его регуляркой:
- `-?` — минус необязателен
- `\d+` — одна или больше цифр
- `(\.\d+)?` — дробная часть необязательна

`fullmatch` — строка должна совпасть **целиком**, не частично. `bool()` превращает результат в `True/False`.

---

```python
with (
    open("soldiers.txt", encoding="utf-8") as fin,
    open("num_counts.txt", "w", encoding="utf-8") as fout,
):
```
Открываем сразу два файла в одном `with`. Когда блок заканчивается — оба автоматически закрываются. `fin` — только чтение, `fout` — запись (`"w"`).

---

```python
results = [
    (i, line.rstrip(), sum(is_num(w) for w in line.split()))
    for i, line in enumerate(fin, 1)
]
```
`enumerate(fin, 1)` — читаем файл строка за строкой, нумерация с 1.

Каждый элемент `results` — это кортеж из трёх вещей:
- `i` — номер строки
- `line.rstrip()` — строка без `\n` на конце
- `sum(is_num(w) for w in line.split())` — делим строку на слова, для каждого слова `is_num()` вернёт `True`/`False`, а `sum()` суммирует их как 1 и 0 — получаем количество чисел

---

```python
fout.writelines(
    f"Строка {i:>2}: {cnt} чисел  | {line}\n" for i, line, cnt in results
)
```
`writelines` принимает генератор строк и пишет их в файл. `{i:>2}` — номер строки выравнивается по правому краю в 2 символа (чтобы `1` и `10` были в столбик).

---

```python
print(open("num_counts.txt", encoding="utf-8").read())
```
Открываем и сразу читаем файл целиком в одну строку — просто для вывода результата в консоль.
